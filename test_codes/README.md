<details>
<summary><b>LoRa (SX1278)</b></summary>

### Структура файлов

* `lora_1/` — **Передатчик**. Отправляет сообщение `"1"` каждые 5 секунд.
* `lora_2/` — **Приёмник**. Постоянно слушает эфир, принимает любые входящие сообщения и выводит их в Serial Monitor.

### Схема подключения (SX1278 → Arduino)

| Пины SX1278 | Пины Arduino |
| :--- | :--- |
| **VCC** | `3.3V` |
| **GND** | `GND` |
| **MISO** | `D12` |
| **MOSI** | `D11` |
| **SCK** | `D13` |
| **NSS** | `D10` |
| **RST** | `D9` |
| **DIO0** | `D2` |

> **Принцип работы:** Передатчик периодически отправляет пакеты данных по радиоканалу LoRa. Приёмник безостановочно сканирует эфир и при успешном получении пакета отправляет его содержимое в последовательный порт (Serial Monitor).

</details>

<details>
<summary><b>GPS (NEO-6M)</b></summary>

### Структура файлов

* `GPS/` — Скетч для работы с модулем геолокации. Трекер обрабатывает данные от спутников и выводит координаты, точное время и другую телеметрию.

### Схема подключения (NEO-6M → Arduino)

| Пины NEO-6M | Пины Arduino |
| :--- | :--- |
| **VCC** | `5V` |
| **GND** | `GND` |
| **TX** | `D10` |
| **RX** | `D11` |

> **Особенности первого запуска (Холодный старт)**
> 
> При первом включении модулю требуется время для поиска альманаха спутников — **обычно от 10 до 40 минут**. 
> * В этот период необходимо обеспечить устройству **постоянное и бесперебойное питание**. 
> * Если питание будет прервано и модуль полностью обесточится (батарейка на самом модуле разряжена или отсутствует), при следующем запуске поиск спутников снова займет столько же времени.
> * Тестирование рекомендуется проводить на улице или у окна с прямой видимостью неба.

</details>
