Этот репозиторий содержит программное обеспечение для управления автономным [водным роботом], включая скрипты для покомпонентного тестирования оборудования и основной код выполнения миссии.

## Моя роль в проекте
* Подбор аппаратных компонентов.
* Проектирование и создание электрической схемы.
* Разработка программного обеспечения (написание кода).
* Сборка и тестирование робота.

## Структура и логика работы
Архитектура проекта подразумевает поэтапную проверку оборудования. **Основной код рекомендуется запускать только после успешного прохождения всех аппаратных тестов.**

### 1. Базовые тесты оборудования
В репозитории представлены скрипты для изолированной проверки каждого модуля:
* **LoRa** — проверка приема и передачи радиосигналов.
* **Компас** — проверка корректности данных о направлении.
* **Моторы** — проверка работоспособности и управления тягой.
* **GPS** — проверка приема координат со спутников.

### 2. Комплексный (промежуточный) тест
Скрипт, объединяющий базовые системы для проверки связи и управления:
* **Телеметрия:** непрерывная отправка данных о положении робота на карте на базовую станцию.
* **Телеуправление:** дистанционное управление моторами через радиоканал LoRa.

### 3. Основной код (Mission Control)
Главный алгоритм, выполняющий целевую задачу робота. Включает в себя следующие этапы:
1. Автономная навигация (движение по воде) в заданную точку на карте.
2. Переход в режим ожидания радиосигнала от базовой станции по LoRa.
3. Выполнение команды: возврат на точку старта или перемещение в новые указанные координаты.
